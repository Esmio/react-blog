------
一款前后端分离，单页面的个人博客
====
所用技术栈
------
前端用React技术栈，React + Redux + React-Router， 
前端后台管理界面antd， 
CSS预处理器用了Sass， 
前后端数据传输用axios库， 
后端用Node,express框架，数据库MongoDB。 

简述
------
第一次从界面设计排版，到前后端编程，到部署上线都独立完成的一个项目（还在等域名备案。。。）

写这个博客，一是想给自己平常所学的东西有个应用发挥的地方。二是平常可以给自己的学习有个记录备忘的地方。

启动
------
npm i

后端启动: node server/app.js

前端启动: npm start
